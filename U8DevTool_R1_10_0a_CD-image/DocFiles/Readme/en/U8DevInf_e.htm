2. Device information file contents

Device information files are classified into the following.

Classification

Extension

Description

DCL file

.dcl

File in which memory map information and information on SFR symbols are defined. DCL files are referenced from the assembler or debugger.

C header file

.h

File in which the SFR symbols for C language are defined.

C-Startup file

.asm

Startup file for C language. Explains how to clear a RAM area to zero, initialize variables, and branch to the main C function.

TRG file

.trg

File for defining debugger operations.

TMI file

.tmi

File for Htu8.
