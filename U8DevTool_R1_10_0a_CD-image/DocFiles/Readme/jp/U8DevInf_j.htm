2.機種情報ファイルの内容

機種情報ファイルには以下のファイルが含まれます。

種類

拡張子

説明

DCLファイル

.dcl

メモリマップの情報やSFRシンボルの情報を定義したファイルです。アセンブラやデバッガから参照されます。

Cヘッダファイル

.h

C言語用のSFRシンボルを定義したファイルです。

Cスタートアップファイル

.asm

C言語用のスタートアップファイルです。RAM領域のゼロクリア、変数の初期化を行い、main関数に分岐します。

TRGファイル

.trg

デバッガの動作を定義するためのファイルです。

TMIファイル

.tmi

HTU8から参照されるファイルです。
