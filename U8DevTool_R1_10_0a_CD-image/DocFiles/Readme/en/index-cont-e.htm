U8 Development Tools

1. Introduction

The U8 Development Tools is a development tool package intended for microcontrollers equipped with 8-bit RISC nX-U8 core.
This document provides information about the products to be set up after installation.

3. Command-line environment

When the U8 Development Tools is installed with the setup program on the setup CD, a group named "nX-U8" is created in the "U8 Tools" folder accessible from the Windows Start menu > "All Programs." Find a "Command-line environment" utility created in this group.

When selected, this Command-line environment utility sets necessary environment variables to open a command prompt environment.
