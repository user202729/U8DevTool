3. Updated Contents

The following updates have been added in this release:

  • The information source of the target microcontroller was changed to the TRG file for each target microcontroller.
  • The function which skips the data of the area which cannot be written in when a program file is loaded was added.
  • ML610Q101/Q102/Q111/Q112/Q355/Q356/Q359/Q360 were supported.
