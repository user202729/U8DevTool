4. 制限事項

現バージョンのＬＣＤイメージツールには、以下の制限事項があります。

(1)     LCDイメージチェックツールにおいて以下の操作を行うと、LCDエミュレータモード/リアルタイムLCDモニタモードでの表示更新速度が低下することがあります。

・        LCD Imageウィンドウの表示倍率を大きくする

・        シルク背景画像を表示する

・        バックライト機能を使用する

(2)     LCDイメージツールで読み込めるセグメント数の制限

LCDイメージアサインメントツールで読み込むイメージファイルに含まれるセグメント数は、最大で16384個までに制限されます。

(3)     LCDイメージアサインメントツールの起動時に選択したLCDパネル割り付けタイプと異なる設定のLCDデータファイルを読み込んだ場合の制限

LCDイメージアサインメントツールにおいて、起動時に選択したLCDパネル割り付けタイプと異なる設定のLCDデータファイルを読み込み、そのまま割り付け設定を変更せずにLCDイメージチェックツールを起動した場合、LCDイメージチェックツールはLCDデータファイルに記述されたLCDパネル割り付けタイプで起動します。

LCDイメージアサインメントツールの起動時に選択したのと同じLCDパネル割り付けタイプでLCDイメージチェックツールを起動したい場合は、以下のうちいずれかをおこなった後でLCDイメージチェックツールを起動するようにしてください。

・        LCDデータファイルを保存する。

・        LCDデータファイルを読み込んだ後でCOM/SEG/DSPRいずれかの割り付け設定を変更する。
