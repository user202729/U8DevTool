4. Restrictions

The current version of LCD Image Tool has the following restriction:

(1) When the following operations are done on LCD Image Check Tool, the display update rate on the Realtime LCD Monitor mode or the LCD Emulator Mode may slow down.

- Enlarge display magnification of a LCD Image window.

- Display the silk background image.

- Use the back light function.

 

(2) The restriction of the number of segments which can load the LCD Image Tool.

The number of segments included in the image file which a LCD Image Assignment Tool can load is restricted by 16384 pieces at the maximum.

 

(3) The restriction when loading the LCD data file of different setting from the display allocation type chosen on the occasion of activation of the LCD Image Assignment Tool

When the LCD data file of different setting from the display allocation type chosen at the time of activation of the LCD Image Assignment Tool was loaded and the LCD Image Check Tool is activated immediately without editing, the LCD Image Check Tool is activated in the display allocation type described in the LCD data file. If you want to activate the LCD Image Check Tool in the same display allocation type as the LCD Image Assignment Tool, you should activate the LCD Image Check Tool after doing either out in following.

- Save the LCD data file.

- Change an assignment within COM/SEG/DSPR after loading the LCD data file.
