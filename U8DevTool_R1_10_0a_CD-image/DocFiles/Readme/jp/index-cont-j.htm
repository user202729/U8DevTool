この文書では、セットアップした製品に関する情報を説明しています。

3.コマンドライン環境

セットアップCDのセットアッププログラムからインストールしたとき、[スタート]→[すべてのプログラム]メニューの下に[U8 Tools]メニューを作成し、その下に[nX-U8]グループを作成します。このグループの中に、[Command-line環境]ユーティリティを登録します。

この[Command-line環境]ユーティリティは、必要な各種環境変数を設定して、コマンドプロンプト環境を開きます。
