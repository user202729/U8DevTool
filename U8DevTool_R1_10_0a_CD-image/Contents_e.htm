Contents of the Setup CD

U8 Development Tools Setup CD contains the following files.

When you click the file name, the PDF file or the folder will open.

 

Folder

Contents

Explanation

\ (Root)

ReleaseNote_e.pdf (English)

ReleaseNote_j.pdf (Japanese)

Release Note.

Important: Read the release note before using this software.

License_e.pdf (English)

License_j.pdf (Japanese)

 

Software License Agreement.

Important: Read the Software License Agreement before using this software.

 

Setup

-

Installer files folder.

See the release note regarding how to install each tools.

 

 

 

U8DevTools

en

SetupU8Dev_e.exe (English)

Installer for U8 Development Tools Software.

This installer will install U8 Development Tools software (IDEU8, CCU8, MACU8, DTU8, LCD Image Tool, etc) and Manuals.

jp

SetupU8Dev_j.exe (Japanese)

FWuEASE

en

SetupFWuEASE_e.exe (English)

 

Installer for Flash Writer for uEASE.

This installer will install FWuEASE.

jp

SetupFWuEASE_j.exe (Japanese)

 

MWuEASE

en

SetupMWuEASE_e.exe (English)

 

Installer for Multiple Flash Writer for uEASE.

This installer will install MWuEASE.

jp

SetupMWuEASE_j.exe (Japanese)

 

U8DevInfo

en

SetupDevInfo_e.exe (English)

 

Installer for Device Information Files.

This installer will install Device Information Files.

Please be sure to install these files after installing the U8 Development Tools Software, FWuEASE or MWuEASE.

jp

SetupDevInfo_j.exe (Japanese)

 

Driver

uEASE_inst_pac

USB device driver files for uEASE and nanoEASE are contained.

DrU8ICE_inst_pac

USB device driver files for Dr.U8 ICE are contained.

DrICE_inst_pac

USB device driver files for Dr.ICE are contained.
